Bank Marketing Prediction API

Solución End-to-End de Machine Learning que predice la suscripción de depósitos a plazo.

Modelo: Random Forest (Scikit-Learn Pipeline).

Ingeniería: API REST con FastAPI y validación Pydantic.

Despliegue: Contenerizado con Docker y desplegado en Render Cloud.

Link a la API: https://api-bank-docker.onrender.com
